# Observability

Set up comprehensive monitoring and logging for your homelab or home server. This section covers the tools and configurations needed to gain visibility into your systems.

## Tools

Install guides for observability stack components:

- **[Alloy]** - Telemetry collector for logs, metrics, traces
- **[Grafana]** - Visualization and dashboards
- **[Loki]** - Log aggregation
- **[Prometheus]** - Metrics storage

## Monitoring Guides

Configuration guides for specific monitoring use cases:

- **[Host & Container Metrics]** - Monitor system and Docker metrics
- **[Docker Logs]** - Collect Docker container logs
- **[System Logs]** - Collect and visualize system logs
- **[Tailscale Metrics]** - Monitor Tailscale clients
